# Getting Started

Clone and initialize

```
git clone git@github.com:lancefisher/hexo-example.git
cd hexo-example
npm install
```

Generate content and run the server

```
hexo generate
hexo server
```

Visit [http://localhost:4000/](http://localhost:4000/)

Add a page:

```
hexo new "Test Page"
```

Modify `Test-Page.md`, then regenerate:

```
hexo generate
```
